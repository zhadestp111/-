## ✨ 功能列表

### 1️⃣ 书签基本管理

| 操作 | 入口 | 说明 |
|------|------|------|
| 添加书签 | 右上角 `➕ 添加` 按钮 | 填写名称、URL、选择分类 |
| 编辑书签 | 右键菜单 → `✏️ 编辑` | 修改名称、URL、分类 |
| 删除书签 | 右键菜单 → `🗑️ 删除` | 删除单条书签 |
| 打开书签 | 单击书签卡片 | 新标签页打开链接 |
| 复制链接 | 右键菜单 → `复制链接` | 复制书签 URL 到剪贴板 |

**添加书签界面：**
```
┌──────────────────────────┐
│  添加书签                │
│  名称: [____________]    │
│  URL:  [____________]    │
│  分类: [▼ 工作       ]   │
│         [取消] [保存]    │
└──────────────────────────┘
```

---

### 2️⃣ 分类管理

侧边栏左侧显示所有分类，支持：

| 操作 | 方式 |
|------|------|
| 新建分类 | 点击侧边栏底部 `＋ 新建分类` |
| 重命名分类 | 双击分类名称，直接输入新名称后按 Enter |
| 删除分类 | 点击分类旁的 `🗑️` 图标 |
| 切换分类 | 点击侧边栏任意分类，右侧内容区自动筛选 |

```
侧边栏示意：
┌─────────────────┐
│ 分类    [＋]    │
│─────────────────│
│ ▶ 全部   (128)  │  ← 当前选中，蓝色高亮
│   工作    (32)  │
│   学习    (45)  │
│   娱乐    (51)  │
└─────────────────┘
```

---

### 3️⃣ 搜索与排序

- **实时搜索**：顶部搜索框输入关键词，书签名称和 URL 同步过滤
- **排序**：点击书签区域右上角的排序按钮，支持按名称/时间/URL 排序
- **视图切换**：`⊞ 网格视图` 和 `≡ 列表视图` 自由切换

---

### 4️⃣ 批量操作

点击 `☐ 多选` 进入批量模式：

```
批量操作工具栏（多选模式下出现）：
┌─────────────────────────────────────────────┐
│ 已选 12 个  [☑全选] [🔄反选]                │
│ [📁移动到分类▼] [✏️批量重命名] [🗑️批量删除]  │
└─────────────────────────────────────────────┘
```

| 功能 | 操作方式 |
|------|----------|
| 批量选择 | 多选模式下单击书签 / 拖拽框选 |
| 全选 / 反选 | 工具栏 `☑ 全选` / `🔄 反选` |
| 批量移动分类 | 工具栏 `📁 移动` → 选择目标分类 |
| 批量重命名 | 工具栏 `✏️ 批量重命名` → 设置规则（前缀/后缀/替换） |
| 批量删除 | 工具栏 `🗑️ 批量删除` → 确认删除 |

**批量重命名规则界面：**
```
┌────────────────────────────────────┐
│  批量重命名                        │
│  规则: [前缀▼] 内容: [____]        │
│  预览: 工作-GitHub → 前缀-工作-... │
│            [取消] [应用]           │
└────────────────────────────────────┘
```

### 5️⃣ 🔴 死链检测

切换到 **死链检测** 标签页，自动检测书签是否可访问：

```
死链检测界面：
┌─────────────────────────────────────────────────────┐
│ 🔴 死链检测                                          │
│ [开始检测]  [使用代理检测]  进度: ████░░░ 45/128    │
├─────────────────────────────────────────────────────┤
│ 🔴 https://old-site.com     → 404 Not Found         │
│ 🟡 https://slow-site.com    → 超时 (>5s)            │
│ 🟢 https://github.com       → 200 OK                │
├─────────────────────────────────────────────────────┤
│ 检测完成：正常 105 / 超时 12 / 死链 11              │
│ [🗑️ 全部删除死链]  [清除标记]                        │
└─────────────────────────────────────────────────────┘
```

| 状态 | 含义 |
|------|------|
| 🟢 绿色 | 链接正常可访问（HTTP 2xx） |
| 🟡 黄色 | 响应超时或重定向 |
| 🔴 红色 | 链接无法访问（4xx/5xx/无响应） |

- **代理模式**：点击 `使用代理检测` 可通过本地代理（127.0.0.1:7890）检测被墙网站
- **停止检测**：检测进行中可随时点击 `停止`
- **批量清理**：检测后点击 `全部删除死链` 一键清理所有失效书签

### 7️⃣ 🌐 域名管理

切换到 **域名管理** 标签页，按网站域名聚合查看所有书签：

```
域名管理界面：
┌─────────────────────────────────────────────────────┐
│ 🌐 域名分组       [☐多选] [🌐按域名自动分类]        │
├─────────────────────────────────────────────────────┤
│ ▼ github.com (8)                                    │
│   ☐ GitHub         ☐ GitHub Issues   ☐ GitHub PR   │
│ ▶ bilibili.com (5)                                  │
```

- **折叠/展开**：点击域名行展开该域名下的所有书签

- **域名批量操作**：选中书签后可批量移动、重命名、删除

### 8️⃣ 导入 / 导出

#### 导入

顶部栏点击 `📥 导入`：

```
导入界面：
┌──────────────────────────────────────────┐
│  导入书签                                │
│  [选择文件]  支持 .html / .json          │
│  ─ 或直接粘贴内容 ─────────────────────  │
│  [___粘贴书签文本/JSON内容___]           │
│  [导入粘贴的内容]                        │
└──────────────────────────────────────────┘
```

| 格式 | 说明 |
|------|------|
| HTML | 浏览器导出的书签 HTML 文件（Chrome/Edge/Firefox 均支持） |
| JSON | 本工具导出的 JSON 格式，完整保留分类信息 |
| 文本粘贴 | 直接粘贴 URL 列表或 JSON 内容 |

#### 导出

顶部栏提供两种导出方式：

| 按钮 | 格式 | 用途 |
|------|------|------|
| `📤 导出HTML` | 浏览器标准书签 HTML | 导入 Chrome/Edge/Firefox |
| `📋 导出JSON` | 含分类信息的 JSON | 备份或在本工具间迁移 |

---

### 9️⃣ 数据存储

- 所有书签和分类**保存在浏览器 LocalStorage**，关闭页面数据不丢失
- 定期使用 `导出JSON` 功能备份数据，防止浏览器清理存储

## 🚀 快速开始

1. 下载 `书签管理器.html` 到本地
2. 双击文件，在浏览器中打开
3. 点击右上角 `➕ 添加` 开始添加书签，或点击 `📥 导入` 导入已有书签

> 💡 **提示**：建议将文件固定在浏览器收藏夹，方便随时访问。
